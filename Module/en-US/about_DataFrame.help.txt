SHORT DESCRIPTION
    Cmdlets for Microsoft.Data.Analysis.DataFrame

LONG DESCRIPTION
    The module provides the following commands:

        Import-DataFrame
        Export-DataFrame
        Read-DataFrame

        New-DataFrame
        Out-DataFrame
        Add-DataFrameRow

        New-BooleanColumn
        New-ByteColumn
        New-CharColumn
        New-DateTimeColumn
        New-DecimalColumn
        New-DoubleColumn
        New-Int16Column
        New-Int32Column
        New-Int64Column
        New-SingleColumn
        New-StringColumn
